# 关于本站：Welcome to my digital garden

Hello, World! 我是 Bai。

这里是我的个人博客，也是我在广阔互联网上亲手搭建的一座“数字花园”。作为一个每天与代码和系统打交道的 CS 人，我把这里当作技术沉淀的自留地，以及折腾各种新奇想法的试验田。

## 💡 起因：为什么要做这个网站？

在建立这个博客之前，我的状态可能和大多数开发者一样：技术笔记散落在各个云文档里，重装双系统或者配置 Java/C++ 环境踩过的坑，下次遇到还要重新百度；刚刚调校好的 CS2 外设参数和 800 DPI 的手感，过阵子可能就忘了。

我需要一个集中的、完全属于我自己的地方。但我不想用 WordPress 那种臃肿的系统，也不想套用千篇一律的现成模板。我想要的，是一个**足够纯粹、专注于 Markdown 内容本身，且极度轻量**的平台。

于是，这个基于原生前端技术和 Fetch API 的静态博客诞生了。

## 🛠️ 经过：与代码和审美的斗智斗勇

这个网站的开发过程，也是我探索现代 AI 辅助编程（AI-Driven Development）的实践。在 Trae 等强大 AI 工具的加持下，我跳过了繁琐的底层轮子，把绝大部分精力都集中在了“架构设计”和“UI 调教”上。

如果你是一路看过来的访客，你会发现这个页面的 UI 经历了无数次“重构”：
- **从简陋到沉浸**：抛弃了最初的呆板列表，尝试过全屏亚克力毛玻璃，最终定格在现在这种“克制、主次分明”的极简深色主题。
- **与 CSS 的死磕**：为了让阅读体验达到极致，我强行覆写了 Waline 评论区的臃肿样式把它“拍扁”；为了在屏幕上呈现出最优雅的排版，我在字体栈（Font Stack）里反复横跳，硬是把 Windows 下发虚发粗的宋体，调教成了现在这种“标题学术、正文清晰”的高级感。

每一次按下 `git push`，看着页面变得更精致一点，那种属于程序员的成就感是无与伦比的。
